# Proyecto Individual - Página Web Personal

**URL del proyecto hospedado:** https://vannkce.github.io/BlogPersonal/ 

**Descripción:**  
Este proyecto es una página web personal que permite explorar secciones como Hobbies, Mi Blog, Música, Películas y Conectemos. Incluye interacción con el usuario mediante JavaScript, envío y almacenamiento de mensajes en MySQL a través de un servidor en Python, y cambios dinámicos en la interfaz según la interacción del usuario. El sitio busca mostrar un espacio creativo y personal, combinando diseño y funcionalidad.

**Lenguajes y tecnologías utilizadas:**  
- HTML  
- CSS  
- JavaScript  
- Python (WSGI server)  
- SQL (MySQL)
